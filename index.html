Python

Multilayer Perceptron Optimization for Minimized Error Probabilistic Classification

Development and optimization of a series of Multilayer Perceptrons (MLPs) for classifying a 3-dimensional real-valued random vector into one of four classes with uniform priors.The objective is to approximate class label posteriors effectively and implement a MAP classification rule using the trained MLPs to achieve the lowest possible probability of error. MLPs are trained using maximum likelihood parameter estimation, equivalent to minimizing the average cross-entropy loss, to approximate the class label posteriors effectively.The trained MLPs are then used to approximate a MAP classification rule, aiming to minimize the expected loss with 0-1 loss assignments for correct and incorrect decisions. The MLPs feature one hidden layer of perceptrons, with the best number determined through cross-validation. K-fold cross-validation and GridSearchCV are employed to identify the optimal number of perceptrons and regularization strength, tailored to the dataset size and complexity.

Bayesian Vehicle Localization Optimization in 2D space via Range Measurements and MAP Estimation

Application of MAP estimation in navigation and spatial data analysis, aimed at enhancing location accuracy in complex environments.The model localizes a vehicle in a 2D state space, combining range measurements to multiple landmarks with pre-existing positional knowledge to accurately estimate the vehicle's most probable location. The model estimates the most probable (MAP) position of the vehicle based on noisy distance measurements to landmarks and prior knowledge of the vehicle's position,effectively navigating environmental complexities and measurement inaccuracies.
